# 双栈模拟队列

## 思路：栈 1 用作入队，栈 2 用作出队，当栈 2 为空时，将栈 1 全部压栈到栈 2，栈 2 在出栈(即出队列)

## 代码

```js
let stack1 = [],
  stack2 = [];
function push(node) {
  stack1.push(node);
}
function pop() {
  if (stack2.length === 0) {
    if (stack1.length === 0) {
      return null;
    } else {
      let len = stack1.length;
      for (let i = 0; i < len; i++) {
        stack2.push(stack1.pop());
      }
      return stack2.pop();
    }
  } else {
    return stack2.pop();
  }
}
```
