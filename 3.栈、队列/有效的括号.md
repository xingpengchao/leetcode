# 有效的括号

## > 富图手撕代码

## 思路：只处理左半边括号

## 代码

```js
const isValid = (s) => {
  let stack = [];
  for (let i = 0; i < s.length; i++) {
    if (s[i] === "(") {
      stack.push(")");
    } else if (s[i] === "[") {
      stack.push("]");
    } else if (s[i] === "{") {
      stack.push("}");
    } else if (stack.pop() !== s[i]) {
      return false;
    }
  }
  return !stack.length;
};
```
