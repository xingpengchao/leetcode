# 验证回文串

## 来源

> leetcode: [传送门](https://leetcode-cn.com/problems/compress-string-lcci/)

## 难度类型：Easy

## 代码

```js
var compressString = function (S) {
  let p = 0;
  let count = 1;
  let res = [];
  while (p <= S.length - 1) {
    if (S[p] === S[p + 1]) {
      count++;
    } else {
      res.push(S[p]);
      res.push(count);
      count = 1;
    }
    p++;
  }
  if (res.length >= S.length) return S;
  return res.join("");
};
```
